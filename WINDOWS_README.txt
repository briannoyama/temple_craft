To set up in windows install:

Python 3.5+

Download Build tools for Visual Studios 17 -https://www.visualstudio.com/downloads/#build-tools-for-visual-studio-2017

SDL2 for Windows Visual Studios version
-> Install SDL2, SDL2_image, and SDL2_mixer into sensible location.
-> From include folders copy contents into C:\Development\SDL2.
-> From lib folders copy contents into C:\Development\SDL2_lib.

Add C:\Development\SDL2 into PATH.

YAY!
